# Darts Homes

This application is created with the goal of creating a game for practicing darts at home.

## How to run

```bash
yarn
yarn start:build
```

## How to develop

```bash
yarn
yarn dev
```

## How to testing

```bash
yarn test
```

```bash
yarn cypress:ci
```

## How to deploy

When you push to the main branch with pull request, the application will be deployed by Vercel.
